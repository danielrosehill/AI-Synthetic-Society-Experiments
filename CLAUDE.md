# AI-Synthetic-Society-Experiments

## Purpose

This repository is a curated collection of resources about projects that have created synthetic societies with AI to explore multi-agent interaction at scale.

## Formatting

- Use shields.io badges for GitHub repos (stars, last commit)
- Organize projects by category/section
- Keep descriptions concise (1-2 sentences)
- Sort projects alphabetically within each section
- Include author links where available
