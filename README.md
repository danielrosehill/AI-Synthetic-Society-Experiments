# AI Synthetic Society Experiments

A curated collection of resources about projects that use agentic AI — including at massive scale — as a proxy to model real sociological and political challenges.

## Scope

**In scope:** Simulations of agent socialization, multi-agent interaction, emergence of deviant behavior, emergent intelligence, and synthetic population modeling for sociological/political insight.

**Out of scope:** General-purpose multi-agent tooling and frameworks that are not specifically focused on simulating social dynamics.

## Foundational Research

Seminal projects and implementations that established the field of AI synthetic societies.

### [Generative Agents: Interactive Simulacra of Human Behavior](https://github.com/joonspk-research/generative_agents)

![GitHub stars](https://img.shields.io/github/stars/joonspk-research/generative_agents?style=social) ![GitHub last commit](https://img.shields.io/github/last-commit/joonspk-research/generative_agents)

The Stanford research project that introduced generative agents — computational software agents that simulate believable human behavior. Agents remember, reflect, and interact in a sandbox environment, producing emergent social behaviors.

**Author:** [joonspk-research](https://github.com/joonspk-research)

---

### [Smallville](https://github.com/nmatter1/smallville)

![GitHub stars](https://img.shields.io/github/stars/nmatter1/smallville?style=social) ![GitHub last commit](https://img.shields.io/github/last-commit/nmatter1/smallville)

A video game implementation of the Generative Agents paper, bringing the "Interactive Simulacra of Human Behavior" concept into a playable environment.

**Language:** Java

**Author:** [nmatter1](https://github.com/nmatter1)

---

### [Human-Simulacra](https://github.com/hasakiXie123/Human-Simulacra)

![GitHub stars](https://img.shields.io/github/stars/hasakiXie123/Human-Simulacra?style=social) ![GitHub last commit](https://img.shields.io/github/last-commit/hasakiXie123/Human-Simulacra)

Official repository for the ICLR 2025 paper "Human Simulacra: Benchmarking the Personification of Large Language Models." Evaluates how well LLMs can simulate individual human personalities and behaviors.

**Language:** Python

**Author:** [hasakiXie123](https://github.com/hasakiXie123)

---

### [LLM Ant Farm](https://github.com/grahamhome/llm-ant-farm)

![GitHub stars](https://img.shields.io/github/stars/grahamhome/llm-ant-farm?style=social) ![GitHub last commit](https://img.shields.io/github/last-commit/grahamhome/llm-ant-farm)

A Python reimplementation of the Stanford Generative Agents experiment using a local LLM. Demonstrates that synthetic society simulations can run without cloud API dependencies.

**Language:** Python

**Author:** [grahamhome](https://github.com/grahamhome)

## Predictive Tooling

Tools that use multi-agent society simulations specifically for prediction and forecasting.

### [MiroFish](https://github.com/666ghj/MiroFish)

![GitHub stars](https://img.shields.io/github/stars/666ghj/MiroFish?style=social) ![GitHub last commit](https://img.shields.io/github/last-commit/666ghj/MiroFish)

A swarm intelligence prediction engine that constructs parallel digital worlds from real-world seed information (news, policy drafts, financial signals), then populates them with thousands of agents with independent personalities and long-term memory. Agents interact across simulated social platforms, and their emergent opinion shifts and coalition formations are analyzed to produce structured prediction reports. Built on the OASIS simulation engine, scaling up to one million agents.

**Language:** Python

**Author:** [666ghj](https://github.com/666ghj)
